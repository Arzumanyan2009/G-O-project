Git bash

հրամաններ

Որպեսզի githubը,այսպես ասած միացնենք մեր ստեղծած ռեպոին պետք է օգտվել git clone հրամանից։

Ռեպոյում ֆայլ ստեղծելու համար պետք է օգտվել touch "ֆայլի անուն" հրամանից։

Git add հրամանը գործողությունների շղթայի առաջին հրամանն է, որը Git-ին հանձնարարում է «պահել» նախագծի ներկա վիճակի պատկերը կատարման պատմության մեջ:

Պարտավորությունները ստեղծվում են git commit հրամանի միջոցով, որը լուսանկարում է նախագծի վիճակը տվյալ պահին:

Git push-ը կոնսոլային հրաման է, որը տեղական պահոցում կատարված փոփոխությունները մղում է gitnub պահոց:
